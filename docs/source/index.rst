Mofa documentation
###############################

Mofa is a moderation and feedback assistent. Mofa uses Learning Locker to automatically moderate and give feedback to Moodle users.
If you are installing Mofa, refer to the SysAdmin Getting Started. If you are a teacher using Mofa, refer to the User Getting Started.
